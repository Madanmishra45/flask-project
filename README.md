# Flask Student Registration API

This project is a simple **Student Registration REST API** built using **Python Flask**.
It allows users to create, view, update, and delete student records using HTTP requests.

---

## 🚀 Features

* Get all students
* Get a single student by ID
* Add a new student
* Update student details
* Delete a student
* Delete all students

---

## 🛠 Technologies Used

* Python
* Flask
* REST API
* JSON

---

## 📂 Project Structure

```
flask-project
│
└── app.py
```

---

## ▶️ How to Run the Project

1. Install Flask

```
pip install flask
```

2. Run the Flask application

```
python app.py
```

3. Open your browser and go to:

```
http://127.0.0.1:5000/students
```

---

## 📡 API Endpoints

| Method      | Endpoint         | Description         |
| ----------- | ---------------- | ------------------- |
| GET         | `/students`      | Get all students    |
| GET         | `/students/<id>` | Get student by ID   |
| POST        | `/students`      | Add new student     |
| PUT / PATCH | `/students/<id>` | Update student      |
| DELETE      | `/students/<id>` | Delete student      |
| DELETE      | `/students`      | Delete all students |
